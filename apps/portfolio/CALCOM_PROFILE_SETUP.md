### Calendar Sync Troubleshooting

If events aren't blocking time slots:

1. **Verify all calendars are checked** under "Check for conflicts"
2. **Ensure "Requires confirmation" is OFF** in event type settings
3. **Mark calendar events as "Busy"** (not "Free")
4. **Disconnect and reconnect** calendars if needed
5. **Wait 5-10 minutes** for sync after making changes

## ğŸ› Troubleshooting

### Issue: Calendar Events Not Blocking Time

**Solution**:
1. Check "Requires confirmation" is OFF
2. Verify all calendars are checked under "Check for conflicts"
3. Ensure events in calendar are marked as "Busy"
4. Disconnect and reconnect Google Calendar
5. Wait 10 minutes for sync

### Issue: No Confirmation Email

**Solution**:
1. Check spam/junk folder
2. Verify email in Cal.com settings
3. Check notification settings are enabled
4. Test with different email address

### Issue: Widget Not Loading on Portfolio

**Solution**:
1. Verify `.env.local` has correct values
2. Restart development server
3. Check browser console for errors
4. Clear browser cache
5. Try incognito/private mode

### Issue: Wrong Timezone

**Solution**:
1. Update timezone in Cal.com Settings â†’ Profile
2. Clear browser cache
3. Refresh booking page
4. Verify timezone in confirmation emails
