## üé® Layout & Design Specifications

### **Fonts**

**Recommended Fonts:**
- **Sans-serif**: Arial, Calibri, Lato, Source Sans Pro, Helvetica
- **Serif**: Times New Roman, Cambria, Georgia
- **Modern**: Inter, Roboto (web-friendly)

**Font Sizes:**
- **Body text**: 10-12pt (11pt is optimal)
- **Your name**: 18-24pt
- **Section headings**: 14-16pt
- **Job titles/company names**: 11-13pt

**Font Colors:**
- Primary: Black (#000000)
- Accent: Dark gray (#333333) or navy (#1a1a2e)
- Limit to 1-2 colors maximum

### **Margins**

**Standard margins: 1 inch (2.54cm) on all sides**
- **Acceptable range**: 0.75" - 1"
- **Minimum**: Never below 0.5"
- **Reason**: Proper whitespace prevents ATS parsing errors

### **Spacing**

**Line Spacing:**
- **Within sections**: 1.0 - 1.15 (single spacing)
- **Between sections**: 2-3 line breaks
- **Paragraph spacing**: 6-12pt after paragraphs

**Best Practice:**
- Use consistent spacing throughout
- Add extra space between major sections
- Avoid cramming text to fit one page

### **Alignment**

- **Left-align** all text (most ATS-friendly)
- **Never use** justified text (creates irregular spacing)
- **Avoid** centering body text

## üéØ Section-Specific Best Practices

### **1. Contact Information**

**Format:**
```
Your Name (18-24pt, bold)
Email | Phone | Location | LinkedIn | Portfolio
```

**Tips:**
- Place at top, left-aligned
- Use simple bullet separator (|) or line breaks
- Include LinkedIn URL (keyword-rich profile)
- Add portfolio/GitHub for tech roles
- **Never** put in header/footer

### **2. Professional Summary**

**Length**: 3-5 sentences (50-100 words)
**Focus**: Keywords + achievements + value proposition

**Example Structure:**
"[Job Title] with [X] years of experience in [Industry/Skills]. Proven track record of [Achievement]. Expert in [Key Skills]. Seeking to leverage [Skills] to [Goal]."

### **3. Core Competencies / Skills**

**Format:**
- Bullet points or comma-separated
- 15-20 relevant skills
- Mix of hard and soft skills
- Include exact keywords from job description

**Categories to include:**
- Technical Skills
- Tools & Technologies
- Methodologies
- Industry Knowledge
- Soft Skills (limited)

### **4. Work Experience**

**Format for each job:**
```
Job Title                                    Month Year - Month Year
Company Name | Location
- Achievement-focused bullet points (4-6 bullets)
- Start with action verbs
- Include metrics and results
```

**Bullet Point Formula:**
"[Action Verb] + [Task] + [Result/Impact] + [Metric]"

**Example:**
"Led cross-functional team of 8 developers, delivering 5 major features ahead of schedule, resulting in 40% increase in user engagement"

### **5. Education**

**Format:**
```
Degree Name, Major
Institution Name | Location | Graduation Year
```

**Tips:**
- List most recent first
- Include GPA if >3.5 (recent grads)
- Relevant coursework for entry-level

### **6. Projects (Optional but Recommended)**

**Format:**
- Project name (bold)
- Brief description (1-2 lines)
- Technologies used
- Achievements/outcomes

### **7. Certifications**

**Format:**
```
Certification Name | Issuing Organization | Year
```

**Tips:**
- List relevant certifications only
- Include expiration dates if applicable
- Prioritize industry-recognized certs

## üîç ATS Optimization Checklist

### **Keywords**

- [ ] Include exact keywords from job description
- [ ] Use industry-standard terminology
- [ ] Spell out acronyms first (e.g., "Search Engine Optimization (SEO)")
- [ ] Include both acronyms and full terms

- [ ] Ensure PDF is text-based (not image/scanned)

### **Structure**

- [ ] Use standard section headings
- [ ] Single column layout
- [ ] Reverse-chronological order
- [ ] Consistent formatting throughout
- [ ] No graphics or images
- [ ] No tables or text boxes

### **Content**

- [ ] Quantify achievements with metrics
- [ ] Use action verbs (Led, Developed, Achieved, etc.)
- [ ] Tailor to each job application
- [ ] Include relevant skills from job description
- [ ] Professional email address

1. **Customize for each job**: Adjust keywords and skills based on job description

10. **LinkedIn alignment**: Match resume keywords with LinkedIn profile
